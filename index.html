🚗 3. KELAS MOBIL - Named Constructor

Named constructor memberikan nama khusus pada constructor untuk berbagai skenario pembuatan objek.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : SI2A

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized Constructor
  Mobil(this.merk, this.tahun);

  // Named Constructor: Mobil Baru
  Mobil.baru() {
    merk = 'Nissan Skyline GT-R R34';
    tahun = 2002;
  }

  // Named Constructor: dari JSON
  Mobil.dariJson(Map<String, dynamic> json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }
}

🎨 4. KELAS WARNA - Constant Constructor

Constant constructor membuat objek yang tidak bisa diubah (immutable) dan dioptimasi memory.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : SI2A

class Warna {
  final int r, g, b;

  // Constant Constructor
  const Warna(this.r, this.g, this.b);

  @override
  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const warnaMerah = Warna(255, 0, 0);
  const warnaHijau = Warna(0, 255, 0);
  
  print(warnaMerah);
}

📐 5. KELAS TITIK - Initializer List

Initializer list mengeksekusi kode sebelum constructor body untuk validasi atau manipulasi nilai awal.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : SI2A

class Titik {
  final double x, y;

  // Initializer list untuk memastikan koordinat bernilai absolut
  Titik(double x, double y)
      : x = x.abs(),
        y = y.abs() {
    print('Titik (${this.x}, ${this.y}) berhasil dibuat.');
  }
}

void main() {
  Titik titik1 = Titik(5.0, 7.0);
  Titik titik2 = Titik(-3.5, -8.0);
}

👨‍🎓 6. KELAS MAHASISWA - Named Constructor

class Mahasiswa {

  String kelas;

  // Default Constructor
  Mahasiswa(this.nama, this.nim, this.jurusan, this.kelas);

👤 7. KELAS PERSON - Enkapsulasi Dasar

Enkapsulasi menyembunyikan data internal dan menyediakan akses melalui getter/setter dengan validasi.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : SI2A

class Person {
  String _nama; // Private field
  int _umur;    // Private field

  Person(this._nama, this._umur);

  // Getter
  String get nama => _nama;
  int get umur => _umur;

  // Setter dengan validasi
  set nama(String namaBaru) {
    if (namaBaru.isNotEmpty) _nama = namaBaru;
  }

  set umur(int umurBaru) {
    if (umurBaru > 0 && umurBaru < 150) _umur = umurBaru;
  }
}

🏦 10. KELAS REKENING BANK - Simulasi Transaksi

Enkapsulasi dengan metode deposit, tarik tunai, dan transfer dengan validasi saldo.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : SI2A

class RekeningBank {
  String _nomorRekening;
  double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jumlah) {
    if (jumlah > 0) _saldo += jumlah;
  }

  void tarik(double jumlah) {
    if (jumlah > 0 && _saldo >= jumlah) _saldo -= jumlah;
  }

  void transfer(RekeningBank tujuan, double jumlah) {
    if (jumlah > 0 && _saldo >= jumlah) {
      _saldo -= jumlah;
      tujuan._saldo += jumlah;
    }
  }
}
